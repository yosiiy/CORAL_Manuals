# Coral Sports (ITSF) 利用規約 要約

**元URL**: `https://app.tablesoccer.org/terms`
**最終確認日**: 2026年3月17日

国際テーブルサッカー連盟（ITSF）が提供するデジタルサービス「Coral」の利用規約の要約です。

## 1. アカウント登録と年齢制限
- アカウント作成時は正確な情報を提供すること（1人で複数アカウントの作成は禁止）。
- **年齢制限**: 16歳以上（または居住国の成人年齢以上）であること。16歳未満の場合は親や保護者の同意と監督が必要。

## 2. アカウント管理とセキュリティ
- パスワード等の管理責任はユーザーにあり、他者への譲渡は禁止。
- 規約違反や虚偽の情報登録があった場合、ITSFはアカウントを停止・削除する権利を持つ。

## 3. 支払いと返金について（Payment Policy）
- Coral自体は利用料を請求しない。プラットフォーム上で発生する支払いは「ユーザーと各組織（クラブ等）」間の直接取引となる。
- **返金**: Coralは決済の仲介のみであり、返金対応は行わない。返金が必要な場合は直接その組織に連絡すること。

## 4. コンテンツと知的財産権
- **Coralのコンテンツ**: プラットフォーム上のデータやロゴはITSFが所有し、無断の商業利用は禁止。
- **ユーザーの投稿**: ユーザーが投稿したデータの著作権はユーザーに残るが、ITSFに対してそれらを世界中で無償利用・表示・改変する無期限のライセンスを付与することになる。

## 5. 肖像権の許諾（重要: 第7.7条）
- Coralに登録しITSFの大会に参加することで、自身の**画像、音声、競技パフォーマンス**が、ITSFやそのパートナーによって（テレビ、YouTube配信、プロモーション等で）無償かつ無制限に使用されることに同意したものとみなされる。

## 6. 免責事項・法的対応
- サービスは「現状有姿（As is）」で提供され、不具合やデータ消失に対する責任をITSFは負わない。
- 紛争が生じた場合は**フランスの法律**が適用され、ナント行政裁判所が管轄となる。また、言語間で解釈の相違があった場合は**英語版の規約が優先**される。
