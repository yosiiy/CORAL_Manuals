# 📘 CORAL (Table Soccer) 基本操作マニュアル

**URL**: `https://app.tablesoccer.asia/profile/`
**対象アカウント**: Hiroshi Yoshida 様

---

## 1. 左側メニュー（サイドバー）
画面左側には、アカウント全体を管理するためのメニューが並んでいます。
- **Profile（プロフィール）**: 今開いているメイン画面です。
- **Personal Info（個人情報）**: 住所や連絡先などの個人設定を変更できます。
- **Accounting（会計）**: 参加費やメンバーシップの支払い状況などを確認できます。
- **My Team（マイチーム）**: チーム戦などの所属チーム情報を管理します。
- **Settings（設定）**: アプリの通知設定などを変更します。
- **Sign Out（サインアウト）**: ログアウトします。

## 2. プロフィール基本情報（画面中央上部）
Hiroshi Yoshidaの基本データが表示されています。
- **Coral ID**: 大会エントリーなどで使う固有のIDです（現在は `6R3V4M`）。
- **ライセンス・スキル**: 国際ライセンス番号やスキルレベル（Beginner JTSFなど）が確認できます。
- 「**Join club**」ボタンから、新しいクラブに所属申請が可能です。

## 3. 各種ステータスと支払い（タブ部分）
所属しているクラブや協会の状況が確認できます。
- **Club / National / International memberships**: 所属状況が表示されます。ITSF（国際）などの会費が未払いの場合は、ここに「**Pay（支払う）**」というボタンが表示され、ここから決済へ進めます。

## 4. 大会と成績（画面下部）
- **Upcoming（今後の予定）**: エントリー済みの大会（例：Kansai Open）が表示されます。ここから大会ページへ飛べます。
- **Rankings（ランキング）**: シーズンやルール（Officialなど）を絞り込んで、自分の「Singles（シングルス）」や「Doubles（ダブルス）」のランキングポイントを確認できます。
- **Statistics（統計）**: 過去の試合結果（勝ち・引き分け・負けの数）やゲーム取得状況がグラフで確認できます。
- **Head to Head（対戦成績）**: 右側の検索窓に他のプレイヤー名を入れると、その人との過去の直接対決の成績を見ることができます。
